# SearchNewsApp
Mobile app for searching news
## Features
* View breaking news
* Open articles in a webview
* Add articles to favorites
* Search news by key words
## Technologies
* MVVM architectural pattern
* Kotlin + Coroutines
* ViewBinding
* LiveData
* Paging3 Library
* Retrofit
* Glide
* Navigation Component
* Dagger Hilt
